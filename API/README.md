## Описание методов АПИ и примеры взаимодействия с ними


#### POST /users/login

**Описание**: Авторизация пользователя.

**Параметры запроса**: Объект, содержащий логин и пароль пользователя.

**Формат ответа**: Объект, содержащий токен авторизации.

**Пример запроса**:

```
POST /users/login
```

```json
{
  "login": "user",
  "password": "password"
}
```


**Пример ответа**:

```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpZCI6MSwibG9naW4iOiJ1c2VyIn0.1"
}
```

#### POST /upload/video

**Параметры запроса**: Исходное видео.

**Формат ответа**: Информация о виральных видео.

**Пример запроса**:

```
POST /upload/video
```

```json
{
    "name_video": "5eec6074e4c974e24755138a5ae8aa77.mp4"
}
```
